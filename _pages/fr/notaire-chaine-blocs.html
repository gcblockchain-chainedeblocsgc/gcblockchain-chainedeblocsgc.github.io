Notaire à base de chaine de blocs

Un service fréquemment utilisé pour les documents sensibles est le besoin d'horodatage et de preuve de propriété automatique. Un système capable de classer un document et lui attacher une référence historique peut être une solution sans intervention humaine à ce besoin . Le document peut inclure des formulaires gouvernementaux, un contrat commercial, une image ou une œuvre d'art. Le principal avantage est que les documents, et/ou ses références sont sauvegardés à perpétuité sur la blockchain. Cela signifie que l'empreinte du document existe dans le cyberespace même si la copie physique disparaît.

Le but

Développer un système web pour télécharger un document d'une taille maximale donnée et avoir un reçu de la preuve de cet acte. Un horodatage et un hachage (représentation cryptographique du document) doivent être inclus.

The system will also provide a way to prove that the file has been notarized or not. This solution must be designed / implemented using Blockchain Technology thats not include this function as part of it's own services.
