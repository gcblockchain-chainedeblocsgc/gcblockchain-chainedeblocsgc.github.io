entitésGC

entitésGC est un projet visant à partager une base de données centrale d’information sur toutes les entités du gouvernement du Canada (les ministères, agences, et sociétés d’État de 1867 à aujourd'hui) parmi les ministères du GC et le public (sur un portail du Gouvernement ouvert). Justification : actuellement, de nombreuses entités du GC assignent des ressources ÉTP au maintien de leurs propres bases de données pour leurs activités spécifiques. Cependant, dans presque tous les cas, les renseignements de base pourraient être créés et mis à jour à partir d'une seule source centrale (Bureau du Conseil Privé ou Secrétariat du Conseil du Trésor). Les problèmes à résoudre incluent la plate-forme, l'interface, les permissions d'édition et les niveaux de sécurité appropriés (puisque certains services peuvent utiliser la base de données pour des activités classifiées / protégées).
